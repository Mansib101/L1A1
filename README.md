# Blog task
1) What are some differences between interfaces and types in TypeScript?
Answer: প্রধান পার্থক্য হলো ইন্টারফেসগুলিকে মার্জ করা যায়, যা ধাপে ধাপে প্রসারণের জন্য উপযোগী। টাইপ অ্যালিয়াসগুলি মার্জ করা যায় না; তাদের সংজ্ঞা চূড়ান্ত। এছাড়াও, জটিল কাঠামোর জন্য টাইপগুলি আরও বহুমুখী, যা তাদের ইউনিয়ন টাইপ এবং টুপল সংজ্ঞায়িত করতে দেয়, যেখানে ইন্টারফেসগুলি প্রধানত অবজেক্ট এবং ক্লাসের আকার নির্ধারণের উপর মনোযোগ দেয়।
2) Explain the difference between any, unknown, and never types in TypeScript.
Answer: তিনটি প্রকার টাইপ কঠোরতার মাত্রা নির্ধারণ করে: any পুরোপুরি টাইপ চেকিং অক্ষম করে, যাচাইকরণ ছাড়াই যেকোনো অপারেশনের অনুমতি দেয়, যা এটিকে সবচেয়ে কম নিরাপদ করে তোলে। unknown হলো any-এর টাইপ-নিরাপদ সংস্করণ; এটি যেকোনো মান গ্রহণ করে তবে এটি ব্যবহার করার আগে সুনির্দিষ্ট টাইপ চেকিং-এর প্রয়োজন হয়, যা আরও নিরাপদ কোডকে উৎসাহিত করে। never এমন মানগুলিকে নির্দেশ করে যা কখনোই ঘটে না বা ফেরত আসে না
